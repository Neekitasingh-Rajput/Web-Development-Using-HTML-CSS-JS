# Web-Development-Using-HTML-CSS-JS
Lab exercises, CAT assessments, and mini web applications from Semester 4: Web Development course at CHRIST (Deemed to be University), covering HTML, CSS, JavaScript, and Django.

# Web-Development-Using-HTML-CSS-JS 🌐

-----

## Project Description

This repository contains the **lab exercises, CAT assessments, and mini web applications** completed for the **Web Development** course during **Semester IV** at **CHRIST (Deemed to be University)**.

The work covers core frontend technologies such as **HTML**, **CSS**, and **JavaScript**, along with an introduction to backend development using **Django**.

-----

## Lab Index

The repository consists of structured lab folders, each demonstrating key web development concepts.

| Folder | Primary Topic | Key Concepts Covered |
| :--- | :--- | :--- |
| **`LAB_01`** | Introduction to HTML & Forms | Basic HTML structure, form elements, input fields |
| **`LAB_02`** | Multi-Page Forms & CSS | Job Portal and School Admission Portal, navigation, form design, CSS selectors |
| **`LAB_03`** | JavaScript Basics | Variables, operators, alerts, basic scripting |
| **`LAB_04`** | DOM Manipulation | Select/Unselect functionality, dynamic UI updates |
| **`LAB_05`** | Form Validation | Dynamic table rows, input validation, user interaction |
| **`LAB_06`** | JavaScript Applications | Calculator, Marks Card, Movie Ticket Booking system |
| **`LAB_07`** | Interactive UI Components | Slideshow, progress bar, event-based styling, validation |
| **`LAB_08`** | Image Gallery | Thumbnail interaction, tooltip display, dynamic image rendering |
| **`LAB_09`** | Countdown Timer | Timer logic, pause/resume functionality, event handling |
| **`LAB_10`** | Django Web Application | URL dispatcher, views, templates, multi-page navigation |

## How to Use This Repository

### Steps to Run

1. Clone the repository:
```bash
git clone https://github.com/Neekitasingh-Rajput/Web-Development-Using-HTML-CSS-JS.git
cd Web-Development-Using-HTML-CSS-JS
```

2. Open any lab folder and run `.html` files directly in a browser.

3. For Django (Lab 10):

```bash
cd LAB_10
python manage.py runserver
```
Then open:

```text
http://127.0.0.1:8000/
```
